Usefulbookmark

Personal bookmark tools to do something useful

Download .zip Download .tar.gz View on GitHub

Usage: drag links below to your bookmarks bar, that's all.

  • Shared2Weibo: 分享当前页面到新浪微博，你可以选择页面上的一张图片进行分享。
  • PixivDownloadHelper: 下载P站静态图大图(单图或多图第一张)和动态图(zip格式)
  • FlashFullPage: Flash视频播放器网页全屏(已测试通过：爱奇艺，B站直播，战旗直播)(用iframe的肯定无效了，B站视频无效)
